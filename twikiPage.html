%CERTIFY% ---+!!%TOPIC% %TOC% %STARTINCLUDE% ---+Introduction This webpage collects all the mc23 Higgs and Di-Higgs Monte Carlo samples available. If you are aware of samples not being tracked in this TWiki, please let the HIGP PMG contacts know (guglielmo.frattari@cern.ch, yanlin.liu@cern.ch), and we will update the list. ---+ Single Higgs samples description For Run 3 single Higgs MC samples, the paradigm with which samples are generated changed. In Run 2 one set of LHE files was generated keeping the Higgs boson undecayed, and the various EVNT were produced by perfoming the Higgs boson decay and the parton showering. Following the discussion hold in the Higgs group, it was decided for Run 3 to generate multiple batches of LHE files - using the same job options - for each Higgs boson production mode. Each batch is then assigned for an Higgs boson decay mode, making the different MC samples statistically independent. For more details, please check [[https://indico.cern.ch/event/1387488/contributions/5838070/attachments/2809604/4903318/20240229_higgsMC.pdf][this presentation]]. Single Higgs boson samples are generated at NLO QCD using PowhegBox v2 for all the production modes, except for tHjb and tWH production, for which Madgraph aMCatNLO is used. For further details, please refer to the job options collected in the table below.
